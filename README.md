# 诊断学基础练习题

本仓库包含一个纯前端的刷题页面（`index.html`），可离线打开练习诊断学基础题库（已整合单选、少量多选）。

## 使用方式
1. 双击或浏览器打开 `index.html`（无需服务器、无需联网）。
2. 左侧控制面板：
   - **篇章选择**：按书本章节筛选题目。
   - **题型选择**：单选 / 多选 / 全部。
   - **练习模式**：
     - *即时验证*：点选即显示对错。
     - *测试验证*：先做完，再点“提交测试”统一判分，可错题重练。
3. 题目区：点击选项作答，已作答会高亮正确/错误。

## 目录
- `index.html`：主页面与题库数据（嵌入式 JSON）。
- `P22ILFf.md`：实验诊断单元测试题源文件。
- `index.html.broken`：先前出问题的备份（可忽略）。

## 维护提示
- 若需追加题目，保持 `DATA` 结构一致（每题包含 `prompt`、`options`、`answers`、`multi`）。
- 新章节标题可随意写，脚本会自动去掉末尾的“单选题/单项选择”等冗余字样以便下拉菜单简洁。
- 修改后建议本地刷新（Cmd/Ctrl+Shift+R）确保缓存更新。
